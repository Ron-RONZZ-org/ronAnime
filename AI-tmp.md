# Init

This repository host my anime.js illustrations to complement my writing

- My style for illustration is
  - minimalist: no extra decoration, no useless animation, just bare minimum to explain the concept
  - low-stimulation, low saturation color palette: with neurodiversity in mind
  - geometric: prefer simple shapes than lots of texts or complex graphics
- this repository will
  - host a bunch of web pages in the `public folder`
  - each web page is an illustration
  - which will be either embedded as iFrame or directly linked to for user to visit on my blog, etc.
Update your `copilot-instructions.md` and memory to save this behaviour as default in this repo
